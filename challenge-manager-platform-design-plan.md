# 챌린지 매니저 플랫폼 디자인 기획서

## 1. 문서 개요

- 문서명: 챌린지 매니저 플랫폼 디자인 기획서
- 작성일: 2026-06-11
- 기준 문서
  - `challenge-manager-platform-plan.md`
  - `challenge-manager-platform-prd.md`
- 목적: 챌린지 매니저 플랫폼의 UX/UI 방향, 정보 구조, 핵심 화면, 시각 설계 원칙을 정의한다.
- 범위: 1차 출시 버전(MVP) 기준

## 2. 디자인 핵심 원칙

- 상태가 먼저 보이는 인터페이스
- 참가자와 운영자의 사용 맥락이 분리된 인터페이스
- 경쟁과 보상은 강하게, 정산과 운영은 명확하게 보이는 인터페이스
- 참가자 화면은 모바일 우선, 운영자 화면은 데스크톱 우선

## 3. 핵심 화면

- 참가자 홈
- 챌린지 탐색
- 챌린지 상세
- 결제
- 내 챌린지
- 인증 제출
- 운영자 대시보드
- 검수 큐
- 정산 관리
- 관리자 모니터링

## 4. 시각 방향

- 기본색: 블루 계열
- 강조색: 오렌지 계열
- 성공색: 그린
- 경고색: 앰버
- 위험색: 레드

## 5. 최종 방향 요약

이 플랫폼의 디자인은 참가자에게는 동기부여와 명확한 진행 상태를, 운영자에게는 빠른 검수와 정산 신뢰성을, 관리자에게는 통제와 추적 가능성을 제공해야 한다.
